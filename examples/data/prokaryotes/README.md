# Prokaryotic Phylogeny

This dataset contains a 15-taxon phylogenetic tree representing major bacterial and archaeal lineages based on 16S rRNA gene sequences.

## Source & References

Topology based on:
- Hug et al. 2016 (Nature Microbiology 1:16048)
- Parks et al. 2018 (Nature Biotechnology 36:996)

Branch lengths represent approximate evolutionary distances in substitutions per site.

## Included Taxa

Bacteria:
- Gamma-proteobacteria: Escherichia, Salmonella, Vibrio, Pseudomonas
- Firmicutes: Bacillus, Clostridium, Staphylococcus, Streptococcus
- Actinobacteria: Mycobacterium
- Epsilon-proteobacteria: Helicobacter
- Cyanobacteria: Cyanobacterium

Archaea:
- Euryarchaeota: Methanobacterium, Halobacterium, Thermoplasma
- Crenarchaeota: Sulfolobus
