# Avian Phylogeny

This dataset contains a 16-taxon phylogenetic tree representing major avian lineages and orders.

## Source & References

Topology based on:
- Jarvis et al. 2014 (Science 346:1320-1331)
- Prum et al. 2015 (Nature 526:569-573)

Branch lengths represent approximate divergence times in units of 0.1 hundred million years (MYA).

## Included Taxa

Paleognathae: Ostrich
Galloanserae: Chicken, Turkey, Duck
Neoaves: Penguin, Flamingo, Pigeon, Parrot, Eagle, Falcon, Owl, Crow, Sparrow, Hummingbird, Woodpecker, Swift
